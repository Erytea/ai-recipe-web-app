О приложении

AI Recipe Bot - создавай персонализированные рецепты с помощью искусственного интеллекта

Что это такое?

AI Recipe Bot - это веб-приложение, которое помогает создавать персональные рецепты на основе фотографий продуктов с точным расчетом калорий, белков, жиров и углеводов.

Распознавание продуктов

Загрузи фото продуктов, и искусственный интеллект автоматически определит, что на изображении. Поддерживаются все популярные форматы изображений.

Точный расчет КБЖУ

Приложение использует базу данных из 500+ продуктов с точными значениями калорий, белков, жиров и углеводов для каждого продукта.

Генерация рецептов

Искусственный интеллект создает рецепты с учетом твоих предпочтений по калориям и макронутриентам. Каждый рецепт уникален и персонализирован.

Планы питания

Создавай полноценные планы питания на день с несколькими приемами пищи, сбалансированные по КБЖУ и распределенные по времени суток.

Технологии

Backend:
  • FastAPI - современный веб-фреймворк
  • Tortoise ORM - асинхронная работа с БД
  • OpenAI API - распознавание изображений и генерация текста
Frontend:
  • Bootstrap 5 - адаптивный дизайн
  • Jinja2 - шаблонизатор
  • Vanilla JavaScript - интерактивность
